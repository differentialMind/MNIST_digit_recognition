# MNIST_digit_recognition
A deep learning model which uses convolution neural networks for MNIST digit recognition, including openCV integration for image capturing and an app interface.

# Installation
clone and install dependencies:
```
git clone https://github.com/yourusername/MNIST_digit_recognition.git
cd MNIST_digit_recognition
pip install -r requirements.txt
```


# Results

The model achieves ~99% accuracy on the test set.
